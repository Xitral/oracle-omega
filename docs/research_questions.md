# Research Questions

## RQ1: Spatial assurance coverage

Which classes of 3D spatial constraints can be represented with a compact, readable rule format?

Initial constraint families:

- protected volume clearance
- approach corridor containment
- alignment tolerance
- relative speed limit
- visibility cone continuity
- surface footprint containment
- tilt limit
- surface infrastructure clearance

## RQ2: Evidence quality

Can users understand why a planned path should be revised more quickly with 3D evidence than with text-only logs?

## RQ3: Runtime cost

Can the prototype evaluate short-horizon planned paths quickly enough for interactive scenario review?

## RQ4: Scenario reproducibility

Can every scenario decision be saved as a small evidence file that can be replayed and inspected later?

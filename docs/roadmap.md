# Roadmap

## Milestone 0: Scaffold

- Add project concept note
- Add Python project config
- Add starter data models
- Add basic spatial checks
- Add example rules and scenarios

## Milestone 1: Local scenario evaluator

- Load YAML scenarios
- Load YAML rules
- Run spatial checks
- Emit an evidence card
- Add unit tests for passing and review scenarios

## Milestone 2: Interactive 3D interface

- Render a scene with a planned path
- Render transparent constraint volumes
- Highlight the first predicted violation
- Show an evidence card beside the scene
- Add replay timeline controls

## Milestone 3: Higher-fidelity simulation

- Replace simple sample paths with generated forecasts
- Add relative motion and attitude updates
- Add uncertainty/fault-case scenario variants

## Milestone 4: Research evaluation

- Build a scenario suite
- Measure coverage, runtime, and false review rate
- Compare 3D evidence with text-only evidence
